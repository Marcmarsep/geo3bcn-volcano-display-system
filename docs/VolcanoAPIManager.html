getMapMetadata(filename, map, successCallback, errorCallback)

Fetches metadata for a specific map associated with a volcano. Sends a POST request with the volcano filename and map name. On success, it logs and forwards the response to the provided success callback. In case of error, it logs the error details and calls the error callback.
Parameters:
Name Type Description
filename string The filename (ID) of the volcano.
map string The specific map to fetch metadata for.

Fetches available maps for a specific volcano. Sends a POST request to the server with the volcano filename and if successful, passes the available map data to the provided success callback. In case of any network or server errors, it handles the error using the provided error callback.
