Class: UIManager

UIManager(config)

UIManager manages the user interface interactions, data presentation, and response to user events. Responsibilities of UIManager include: - Initializing and updating the map view. - Handling user inputs and triggering appropriate actions. - Managing the display of data such as volcanoes, maps, and event trees.

Constructor

new UIManager(config)

Constructs an instance of UIManager. This constructor initializes the UI manager with necessary configurations and dependencies. It sets up the API managers for handling volcano data and map visualizations and defines the protocol, host, web host, and port for network requests.

config Object Configuration object containing necessary parameters.
